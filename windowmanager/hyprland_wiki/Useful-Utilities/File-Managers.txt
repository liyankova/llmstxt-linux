File Managers

GUI

dolphin

: File manager by KDE.

nautilus

: File manager by Gnome.

nautilus-admin-gtk4

: Open files with elevated privileges.

nautilus-image-converter

: Resize and rotate images.

nautilus-open-any-terminal

: Open terminals in selected directory.

nemo

: File manager by Cinnamon.

nemo-fileroller

: File archiver extension.

nemo-terminal

: Embedded terminal window.

pcmanfm-qt

: File manager by LXQt.

pcmanfm

: File manager by LXDE.

thunar

: File manager by XFCE.
