## Quickshell.Io Definitions

Io types

[DataStream](/docs/v0.2.1/types/Quickshell.Io/DataStream)  Data source that can be streamed into a parser.

[DataStreamParser](/docs/v0.2.1/types/Quickshell.Io/DataStreamParser)  Parser for streamed input data.

[FileView](/docs/v0.2.1/types/Quickshell.Io/FileView)  Simple accessor for small files.

[FileViewAdapter](/docs/v0.2.1/types/Quickshell.Io/FileViewAdapter)

[FileViewError](/docs/v0.2.1/types/Quickshell.Io/FileViewError)

[IpcHandler](/docs/v0.2.1/types/Quickshell.Io/IpcHandler)  Handler for IPC message calls.

[JsonAdapter](/docs/v0.2.1/types/Quickshell.Io/JsonAdapter)  FileView adapter for accessing JSON files.

[JsonObject](/docs/v0.2.1/types/Quickshell.Io/JsonObject)

[Process](/docs/v0.2.1/types/Quickshell.Io/Process)  Child process.

[Socket](/docs/v0.2.1/types/Quickshell.Io/Socket)  Unix socket listener.

[SocketServer](/docs/v0.2.1/types/Quickshell.Io/SocketServer)  Unix socket server.

[SplitParser](/docs/v0.2.1/types/Quickshell.Io/SplitParser)  DataStreamParser for delimited data streams.

[StdioCollector](/docs/v0.2.1/types/Quickshell.Io/StdioCollector)  DataStreamParser that collects all output into a buffer
