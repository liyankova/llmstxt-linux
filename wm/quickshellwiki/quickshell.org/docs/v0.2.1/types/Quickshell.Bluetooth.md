## Quickshell.Bluetooth Definitions

Bluetooth API

[Bluetooth](/docs/v0.2.1/types/Quickshell.Bluetooth/Bluetooth)  Bluetooth manager

[BluetoothAdapter](/docs/v0.2.1/types/Quickshell.Bluetooth/BluetoothAdapter)  A Bluetooth adapter

[BluetoothAdapterState](/docs/v0.2.1/types/Quickshell.Bluetooth/BluetoothAdapterState)  Power state of a Bluetooth adapter.

[BluetoothDevice](/docs/v0.2.1/types/Quickshell.Bluetooth/BluetoothDevice)  A tracked Bluetooth device.

[BluetoothDeviceState](/docs/v0.2.1/types/Quickshell.Bluetooth/BluetoothDeviceState)  Connection state of a Bluetooth device.

This module exposes Bluetooth management APIs provided by the BlueZ DBus interface.
Both DBus and BlueZ must be running to use it.

See the [Bluetooth](/docs/v0.2.1/types/Quickshell.Bluetooth/Bluetooth) singleton.
