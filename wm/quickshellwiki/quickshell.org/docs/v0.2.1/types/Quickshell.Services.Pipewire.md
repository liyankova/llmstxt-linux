## Quickshell.Services.Pipewire Definitions

Pipewire API

[Pipewire](/docs/v0.2.1/types/Quickshell.Services.Pipewire/Pipewire)  Contains links to all pipewire objects.

[PwAudioChannel](/docs/v0.2.1/types/Quickshell.Services.Pipewire/PwAudioChannel)  Audio channel of a pipewire node.

[PwLink](/docs/v0.2.1/types/Quickshell.Services.Pipewire/PwLink)  A connection between pipewire nodes.

[PwLinkGroup](/docs/v0.2.1/types/Quickshell.Services.Pipewire/PwLinkGroup)  A group of connections between pipewire nodes.

[PwLinkState](/docs/v0.2.1/types/Quickshell.Services.Pipewire/PwLinkState)  State of a pipewire link.

[PwNode](/docs/v0.2.1/types/Quickshell.Services.Pipewire/PwNode)  A node in the pipewire connection graph.

[PwNodeAudio](/docs/v0.2.1/types/Quickshell.Services.Pipewire/PwNodeAudio)  Audio specific properties of pipewire nodes.

[PwNodeLinkTracker](/docs/v0.2.1/types/Quickshell.Services.Pipewire/PwNodeLinkTracker)  Tracks all link connections to a given node.

[PwNodeType](/docs/v0.2.1/types/Quickshell.Services.Pipewire/PwNodeType)  The type of a pipewire node.

[PwObjectTracker](/docs/v0.2.1/types/Quickshell.Services.Pipewire/PwObjectTracker)  Binds pipewire objects.
