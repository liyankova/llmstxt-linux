## Quickshell.Widgets Definitions

Bundled widgets

[ClippingRectangle](/docs/v0.2.1/types/Quickshell.Widgets/ClippingRectangle)  Rectangle capable of clipping content inside its border.

[ClippingWrapperRectangle](/docs/v0.2.1/types/Quickshell.Widgets/ClippingWrapperRectangle)  ClippingRectangle that handles sizes and positioning for a single visual child.

[IconImage](/docs/v0.2.1/types/Quickshell.Widgets/IconImage)  Image component for displaying widget/icon style images.

[MarginWrapperManager](/docs/v0.2.1/types/Quickshell.Widgets/MarginWrapperManager)  Helper object for applying sizes and margins to a single child item.

[WrapperItem](/docs/v0.2.1/types/Quickshell.Widgets/WrapperItem)  Item that handles sizes and positioning for a single visual child.

[WrapperManager](/docs/v0.2.1/types/Quickshell.Widgets/WrapperManager)  Helper object for creating components with a single visual child.

[WrapperMouseArea](/docs/v0.2.1/types/Quickshell.Widgets/WrapperMouseArea)  MouseArea wrapper item.

[WrapperRectangle](/docs/v0.2.1/types/Quickshell.Widgets/WrapperRectangle)  Rectangle that handles sizes and positioning for a single visual child.
